# Pixiv_Selenium_excel

## 專案說明

此專案使用 Python 的 Selenium 模組，從Pixiv網站爬取圖片和相關資訊，並將數據整理後存儲到 Excel 檔案中。數據包含作品名稱、作者、圖片，幫助用戶了解該地區的房價資訊，便於後續分析。

## 技術關鍵字

- Selenium
- requests
- re
- openpyxl

## 功能

- 爬取圖片數據
- 將數據存入 Excel 檔

## 使用資料

- Pixiv網站

## 環境需求

- Python 3.9.11
- Selenium 4.23.1
- Openpyxl 3.1.5
